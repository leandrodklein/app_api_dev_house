### Inicializando o projeto
- yarn init -y

### Instalando o Express
- yarn add express

### Instalar o Sucrase(usando o import)
- yarn add sucrase

### Instalar o Nodemon
- yarn add nodemon

### Instalar o Mongoose
- yarn add mongoose

### Instalar a biblioteca Multer
- yarn add multer

### Instalar o Cors
- yarn add cors

### Instalar a biblioteca Yup - validações de dados dos formulários

### Rodar o projeto 
- yarn dev
